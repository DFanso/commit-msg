# commit-msg
## Add GROK_API_KEY to system variables


 ## Setup
 ```bash
go run src/main.go --setup --path F:/Git/commit-msg --name commit-msg
```

## Usage

```bash
go run src/main.go --path F:/Git/commit-msg
```
### this will run the commit-msg in the current directory
```bash
go run src/main.go .
```
